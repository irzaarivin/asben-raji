
# Initialization - BackEnd

Hanya membutuhkan beberapa langkah untuk menjalankan aplikasi backend ini.




## Installation

Download seluruh package yang dibutuhkan :

```bash
  npm install
```


## Configuration

Ubah file .env.example ke .env dan ubah isinya sesuai dengan konfigurasi di komputer anda.



## Running Project

Jalankan aplikasi front-end :

```bash
  npm run dev
```
